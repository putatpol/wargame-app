# การทำงานเว็บ
1. หน้าแรกแสดงข้อมูลตัวละคร
- รูป, รายละเอียด, status, skill card

2. เลือกตัวละคร
- เลือกเข้าทีม A or B
- สุ่มตามจำนวนที่ต้องการ
- รีเซทที่เลือกได้
- เมื่อเลือกจะมีโบนัสค่าสถานะของแต่ละเผ่า

3. หน้าเล่นแสดงตัวละครในแต่ละทีม
- การจบเทิร์น, รีเซ็ทเทิร์น
- เพิ่มให้เลือก stat boost สำหรับเผ่า Human ได้ 1 อย่าง
- เพิ่ม status action, buff, debuff แสดงเป็น badge
- ปรับค่า hp manual ได้
- ลบออกจากทีม

4. ระบบการต่อสู้
- เลือกผู้โจมตีและป้องกัน
- แสดงค่า hit on และ def ที่แต่ละตัวต้องทอย (แยกสีตามทีม)
- การ cover ลดค่า def ให้ทอยง่ายขึ้น
- การ counter สลับการโจมตีกลับ จะไม่ลด ap
- การโจมตีของตัว range ตอนโจมตี melee เรทการทอยจะเพิ่มขึ้น
- การใช้สกิลจะลด ap ตามสกิลนั้น (ยังไม่คำนวนดาเมจ, การบัพ)
- การโจมตีปกติมี 3 แบบ โจมตีสำเร็จ, โจมตีคริติคอล(dmg*1.5) และโจมตีพลาด
- ทุกการโจมตีจะลด ap และหากสำเร็จจะคำนวนการลดเลือด
- ข้ามเทิร์นตัวละครได้
- สลับตำแหน่ง dialog ซ้ายขวาได้หากบดบัง content
