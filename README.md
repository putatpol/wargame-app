## Wargame web flow

### 1. หน้าแรกแสดงข้อมูลตัวละคร
```bash
- รูป, รายละเอียด, status, skill card
- มีการแจ้งเตือนทุก action
```

### 2. เลือกตัวละคร
```bash
- เลือกเข้าทีม A or B
- สุ่มตามจำนวนที่ต้องการ
- รีเซทที่เลือกได้
- เมื่อเลือกจะมีโบนัสค่าสถานะของแต่ละเผ่าและตำแหน่ง
  - Tank damage -1 เมื่อโดนโจมตี
  - Human ให้เลือก stat boost ได้ 1 อย่าง
  - Dragonborn แสดงค่าเมินดาเมจจากสถานะ
  - Dwarf HP +2 [pending]
  - Goliath HP +3, Def +1
  - Elf Move +1
  - Lizardfolk Def -1 [pending]
```

### 3. หน้าเล่น
```bash
- แสดงตัวละครในแต่ละทีม
- การจบเทิร์น(เพิ่มจำนวนรอบ และ reset Ap), รีเซ็ทเทิร์น (reset Hp และ Ap)
- เพิ่ม status action, buff, debuff แสดงเป็น badge
- หากกดใช้ action จะลด ap -1
- หากเป็น buf หรือ debuff ให้ลดหรือเพิ่มของ status นั้น [pending]
- แสดงการทอย resist เมื่อเป็น debuff [pending]
- ปรับค่า hp manual ได้
- ลบออกจากทีม
```

### 4. ระบบการต่อสู้
```bash
- เลือกผู้โจมตีและป้องกัน
- แสดงค่า hit on และ def ที่แต่ละตัวต้องทอย (แยกสีตามทีม)
- การโจมตีปกติมี 3 แบบ โจมตีสำเร็จ, โจมตีคริติคอล(dmg\*1.5) และโจมตีพลาด
- ทุกการโจมตีจะลด ap และหากสำเร็จจะคำนวนการลดเลือด
- ทุกการโจมตีจะเพิ่ม On Hit และ Def ของการต่อสู้นั้น [pending]
- ปุ่ม Charge ใช้ได้เฉพาะ Vanguard จะเพิ่ม damage +1 ให้กับการโจมตี
- ปุ่ม Gang จะลดค่า hit on ง่ายขึ้นหากอยู่กับเพื่อนร่วมทีม
- ปุ่ม cover ลดค่า def -2 ให้ทอยง่ายขึ้น
- ปุ่ม counter สลับการโจมตีกลับ จะไม่ลด ap
- ปุ่ม counter มี โจมตีสำเร็จ, โจมตีคริติคอล(dmg\*1.5) และโจมตีพลาด
- ปุ่มการโจมตีของตัว range ตอนโจมตี melee จะเรทการทอย Hit on +4
- ปุ่มการโจมตีแบบ free จะไม่เสียค่า ap
- ปุ่่ม skill จะแสดงการ์ด และรายละเอียด
- การใช้ skill จะลด ap ตาม skill นั้น (ยังไม่คำนวนดาเมจ, การบัพ)
- หากใช้ skill จะแสดง Hit on ของ skill ที่ตัวโจมตี
- ข้ามเทิร์นตัวละครได้
- สลับตำแหน่ง dialog ซ้ายขวาได้หากบดบัง content
- Support มีปุ่มให้ ap กับเพื่อนร่วมทีมได้ครั้งละ 1 ต่อเทิร์น [pending]
```
